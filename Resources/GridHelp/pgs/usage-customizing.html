  Customizing
 

To change the keyboard shortcuts for switching between multiple monitors, do the following:

  • Show Grid's Dock icon through its Preferences.
  • Restart Grid if necessary.
  • Look up the menu commands in the Window menu.
  • Open System Preferences > Keyboard > Keyboard Shortcuts and add your changes to the Application Shortcuts..

For more information about how customizing application shortcuts work please refer to Apple's documentation.
